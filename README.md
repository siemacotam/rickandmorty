## General info

This is the rectuitment task - simple app with Rick and Morty characters.
User can :

- List all characters
- Sort them by part or full name
- Filter them by species (multiple choice)

### Technologies

Project is created with :

- React and Typescript
- Redux + redux/toolkit
- Mui
